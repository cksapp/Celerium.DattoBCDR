# Remove-DattoBCDRBaseURI

## SYNOPSIS
Removes the Datto base URI global variable

## SYNTAX

```powershell
Remove-DattoBCDRBaseURI [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The Remove-DattoBCDRBaseURI cmdlet removes the Datto
base URI global variable

## EXAMPLES

### EXAMPLE 1
```powershell
Remove-DattoBCDRBaseURI
```

Removes the Datto base URI global variable
