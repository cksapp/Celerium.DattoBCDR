# Get-DattoBCDRBaseURI

## SYNOPSIS
Shows the Datto base URI global variable

## SYNTAX

```powershell
Get-DattoBCDRBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-DattoBCDRBaseURI cmdlet shows the Datto
base URI global variable value

## EXAMPLES

### EXAMPLE 1
